# 🏨 The React Hotel

[Live Demo](https://thereacthotel.vercel.app/)

A modern **hotel management web application** built with **React**, **TypeScript**, **Supabase**, and **React Query**.

The app allows hotel staff to manage bookings, cabins, and guests through a clean dashboard interface. It demonstrates modern React development practices including server state management, authentication, and scalable UI architecture.

---

# 🚀 Features

- 🛎️ Authentication system for hotel staff
- 📅 Booking management (create, update, delete bookings)
- 🏡 Cabin management with pricing and capacity
- 👥 Guest information tracking
- 📊 Dashboard analytics showing booking statistics
- 💳 Payment and status tracking
- 🔎 Filter and sort bookings
- 📱 Fully responsive UI
